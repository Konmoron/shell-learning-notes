# 参数传递

在运行shell脚本时，在其后添加参数，shell脚本即可调用这些参数。

调用的格式是：`$number`，其中`number`是参数的索引，从1开始。`$0`是当前脚本的名字。

特殊的调用格式有：

|名称|作用|
|:-:|:-:|
|`$#`|传递参数的个数|
|`$0`|当前脚本的名字|
|`$@`|得到所有参数，将参数作为一个数组|
|`$*`|得到所有参数，将参数作为一个字符串|
|`$$`|脚本运行的当前进程ID号|

例子：

``` 
touch test_Passing_Arguments && vim test_Passing_Arguments
``` 

在test_Passing_Arguments里面输入如下内容：

``` 
\#!/bin/bash

echo -e "\n"

echo "shell 测试参数传递"
echo "测试参数传递，文件名：$0"
echo "测试参数传递，第一个参数：$1"
echo "测试参数传递，第二个参数：$2"
echo "测试参数传递，第三个参数：$3"

echo -e "\n"

echo "参数传递的个数为：$#"

echo -e "\n"

echo "------*的演示-------"
echo "所有的参数为：$*"
        for i in "$*"; do
                echo $i
        done
echo -e "\n"

echo "------@的演示------"
echo "所有的参数为：$@"
        for i in "$@"; do
                echo $i
        done
echo -e "\n"

echo "进程ID为：$$"
```
